## 关于本书

本项目旨在为中文学生提供一本全面的北美互联网实习求职资源手册。在求职过程中，我发现中文学生的求职资源相对匮乏，特别是在互联网行业，因此创建了这本开源指南。

- 全书采用我与周围许多同学的个人经验，帮助中文学生更好地理解北美互联网求职流程。
- 涵盖从准备材料到面试技巧的全方位指导，让求职新手也能轻松上手。
- 提供丰富和权威的的求职工具和资源。

若本书对您有所帮助，请在页面右上角点个 Star :star: 支持一下，谢谢！
